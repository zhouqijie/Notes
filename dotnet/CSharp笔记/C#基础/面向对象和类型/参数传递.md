# 参数传递    

## 按值和按引用传递参数    

结构按值传递，类按引用传递。    

- 结构使用`ref`参数    

结构添加`ref`修饰符可以按引用传递。    

- 类对象使用`ref`参数    

类使用`ref`修饰符，可以传递**引用的引用**。（对应C++中指针的指针）    

> 这样不仅可以修改参数对象的属性，还可以为原来的变量分配一个新对象。    


- 使用`out`参数    

方法返回多个值的三种途径：  
1. 使用元组Turple。    
2. 把这些返回值打包成类或者结构。（CRE:一般命名为"XXResult"）  
3. 使用`out`关键字。      

> 方法返回时必须初始化out参数。
